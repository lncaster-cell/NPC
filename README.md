# NPC Ambient Life v2 (NWN2)

`NPC Ambient Life v2` — это событийная система «живого мира» для NPC в Neverwinter Nights 2.
Она управляет рутинами, переходами между активностями, сном и локальными реакциями на инциденты без постоянного heartbeat/polling на каждом NPC.

## Краткая инструкция по подключению

1. Импортируйте скрипты из `scripts/ambient_life/` в ваш NWN2-модуль.
2. Подключите entry scripts по инструкции из `INSTALLATION.md`.
3. Настройте локальные переменные у NPC/waypoints/areas согласно `docs/TOOLSET_CONTRACT.md`.
4. Запустите базовую проверку (smoke) по `TASKS.md`.
